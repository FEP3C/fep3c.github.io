0.1 介绍

欢迎来到小白电脑扫盲课！这章教程将帮助你了解基本的电脑知识，包括键盘快捷键、键盘上的重要按键、如何选择和使用浏览器、如何避免广告和流氓软件，以及如何安全下载软件。即使你对电脑一无所知，也不用担心，我们会一步一步地带你了解这些知识。

0.2 键盘和键盘快捷键

键盘大家都会用，但是了解一些关键的键是非常重要的。所以这章我们将教授一些键盘基础知识，而不仅仅是使用W A S D 。

0.2.1键盘布局

这是一张常见的108键键盘布局图：

首先，让我们讲解一些冷知识：什么是键盘布局？
最早的键盘布局五花八门，可以说是百花齐放，一个比一个难用。到了1985年，当时电脑界的巨头IBM推出了配套PC的键盘——Model M，这是键盘标准化的重要一步（如下图），此时这个键盘布局除了Windows徽标键，以及中间的delete ，home ，end 等键以外，已经非常接近现在所用的键盘。后来，随着Windows的发展，1998年，正式的PC键盘布局（如上图）正式推出。

0.2.2什么是键盘快捷键

键盘快捷键是一种通过同时按下一个或多个键来快速执行特定操作的方法。它们可以提高工作效率，减少使用鼠标的频率。以下是一些常用的快捷键：

  • 复制：按下 Ctrl + C
  • 粘贴：按下 Ctrl + V
  • 剪切：按下 Ctrl + X
  • 撤销：按下 Ctrl + Z
  • 全选：按下 Ctrl + A
  • 保存：按下 Ctrl + S
  • 调试/刷新：按下F5
  • 切换程序： 按下Alt + Tab

0.2.3 常见键的功能

  • Ctrl（控制键，一般位于键盘最左下角）：用于组合快捷键。
  • Alt（换挡键，一般位于Windows徽标键右边）：用于组合快捷键，切换应用程序。
  • Shift（上档键，在回车键下面）：用于大写字母和特殊符号。
  • Enter（回车键，在上档键上面）：用于确认操作或换行。
  • Backspace（退格键，在+ 键右侧）：用于删除光标前的字符。
  • Delete（删除键，在108键键盘的中间，Insert 键下方）：用于删除光标后的字符或选中的项目。
  • Esc（退出键，键盘最左上角）：用于取消当前操作或关闭弹出窗口。

0.3 选择浏览器

0.3.1 什么是浏览器

浏览器是一种软件，用于访问和查看互联网内容。常见的浏览器包括：

  • 谷歌浏览器（Google Chrome）：速度快，扩展多，适合大多数用户。
  • 微软浏览器（Microsoft Edge）：Windows系统默认浏览器，兼容性好。
  • 火狐浏览器（Mozilla Firefox）：开源、隐私保护好，适合注重隐私的用户。
  • 苹果浏览器（Safari）：苹果设备的默认浏览器，速度快，节能。

在选择浏览器时，我们需要考虑多个方面来评估它的优劣。

首先，我们需要考虑浏览器的性能，即它在加载网页时的速度和流畅度。如果一个浏览器的加载速度慢，用户体验就会受到影响，导致浏览网页变得繁琐。

其次，我们需要考虑浏览器的扩展性。浏览器是否支持各种插件和扩展，能否根据用户的需求进行个性化设置，是选择浏览器时需要考虑的一个重要因素。如果一个浏览器不支持扩展，用户将无法根据自己的需求对浏览器进行个性化设置，从而影响到浏览体验。

除此之外，我们还需要考虑浏览器对用户隐私的保护程度。随着网络技术的发展，用户的个人信息越来越容易被泄露。因此，选择一个浏览器时，需要关注它对用户隐私的保护程度。例如，一些浏览器会采用安全技术来防止恶意软件的攻击，从而保护用户的个人信息不被泄露。

最后，我们需要考虑浏览器的兼容性。浏览器是否能够正常运行我们常用的网站和服务，是否会出现兼容性问题。如果一个浏览器不能正常运行某些网站或服务，那么我们就需要重新选择一个浏览器。

综上所述，选择一个适合自己的浏览器需要综合考虑多个方面。我们需要选择一个性能良好、扩展性强、隐私保护好、兼容性强的浏览器，从而保证我们在浏览网页时能够得到良好的体验。

0.4 避免广告和流氓软件

0.4.1 如何识别广告和流氓软件

广告通常以弹出窗口、横幅或视频的形式出现，内容多为推销产品或服务。流氓软件是指未经用户同意，自动安装并难以卸载的软件。它们非常难以对付，所以需要小心对待。

以上内容中，建议避免安装广告拦截器和不点击可疑链接，但同时提供了其他建议。以下是对每个要点的详细说明：

如何避免

安装广告拦截器

广告拦截器可以帮助屏蔽广告，提高用户体验。但是，使用广告拦截器也可能会影响一些网站的访问，例如电子商务网站或博客。在使用广告拦截器之前，建议先阅读网站的使用条款和隐私政策，以确保使用广告拦截器不会违反相关规定。

以下是几个常用的广告拦截器：

  • AdBlock Plus：支持 Chrome、Firefox、Edge、Safari、Opera 等浏览器。
  • AdGuard：支持 Chrome、Firefox、Edge、Safari、Opera 等浏览器。
  • uBlock Origin：支持 Chrome、Firefox、Edge、Safari、Opera 等浏览器。

安装广告拦截器非常简单。在浏览器中打开“设置”（三个点或者三条横杠），并打开相应的扩展页面（如 AdBlock Plus 的 Edge 扩展页面），点击“添加到 ”按钮即可完成安装。在其他浏览器中，也可以通过相应的应用商店或官方网站下载广告拦截器的插件。

在国内，由于国家防火墙的原因，Chrome的插件商店一般对小白来说难以访问。但是，一旦你用微软浏览器和火狐浏览器，它们的扩展商店是可以直接访问的，而且插件数量和质量和Chrome大差不差。

不点击可疑链接

点击可疑链接可能会导致恶意软件的安装，从而损害计算机的安全。因此，不要随意点击陌生邮件或网页上的链接，特别是不要点击附带下载或安装文件的链接。如果您收到一封看起来可疑的邮件，最好不要点击其中的链接，而是将其标记为垃圾邮件或直接删除。

使用官方渠道下载软件

为了避免下载恶意软件，建议从软件的官方网站或可信的应用商店下载软件。不要从第三方网站或论坛下载软件，因为这些网站上的软件可能存在安全漏洞或病毒。此外，也要避免下载破解软件或盗版软件，因为这些软件可能包含恶意代码。

安装安全软件

安装安全软件可以帮助检测和清除恶意软件。常用的安全软件包括 Windows Defender、火绒安全软件等。在安装安全软件之前，请先备份您的数据，以免意外数据丢失。

以上是几个常见的安全建议。除此之外，还有其他一些建议可以帮助您保护计算机的安全，例如定期更新操作系统和软件、使用强密码、不使用公共无线网络等。

0.5 安全下载软件

下面是每个选项的详细说明：

  1. 官方网站：选择软件的官方网站可以确保下载的软件是最新版本的，并且是由开发人员发布的。官方网站还提供了更好的文档和支持。
  2. 可信应用商店：可信应用商店通常是由著名的软件公司或知名平台运营的，这些应用商店的审核流程较为严格，下载的软件质量也比较有保证。此外，可信应用商店也提供了一些有用的功能，如自动更新、备份等。
  3. 知名包管理器：知名包管理器可以自动管理软件的下载和安装，避免了手动下载和安装软件的繁琐过程。同时，这些包管理器也会提供一些额外的功能，如自动更新、备份等。

需要注意的是，选择下载来源时，应该避免从不知名的网站或非官方网站下载软件。这些网站可能存在恶意软件或病毒，会对计算机造成威胁。因此，建议从官方网站或可信应用商店下载软件。

0.5.2 下载步骤

  1. 打开浏览器：选择你信任的浏览器，如 Google Chrome。
  2. 搜索软件：在搜索引擎中输入你要下载的软件名称和“官方”字样。
  3. 访问官方网站：点击搜索结果中的官方链接。
  4. 找到下载链接：在官方网站上找到并点击下载链接。
  5. 安装软件：下载完成后，双击安装文件，按照提示进行安装。

0.6 其他知识

0.6.1 文件和文件夹

  • 文件：存储数据的单个单位，如文档、图片、视频等。
  • 文件夹：用于组织和存储多个文件。

0.6.2 如何创建文件夹

  1. 在桌面或文件管理器中，右键点击空白区域。
  2. 选择“新建”，然后选择“文件夹”。
  3. 输入文件夹名称，按下 Enter 键。

0.6.3 如何移动文件

  1. 选择文件：点击要移动的文件。
  2. 剪切文件：按下 Ctrl + X 或右键选择“剪切”。
  3. 选择目标文件夹：点击目标文件夹。
  4. 粘贴文件：按下 Ctrl + V 或右键选择“粘贴”。

0.7 什么是终端

1. 简单定义：

终端（Terminal）是一个让你直接与操作系统进行交互的工具。通过终端，你可以输入命令，操作系统会执行这些命令并显示结果。

2. 类比解释：

终端就像是一个对话框，你可以通过它与电脑“对话”。当你输入一条命令并按下回车键时，就好比你向电脑说了一句话，然后电脑会根据你的指令去做事情并告诉你结果。

3. 图形界面与命令行界面的对比：

通常我们使用电脑的方式是通过图形界面（GUI），比如点击图标打开应用程序。而终端是一种命令行界面（CLI），你需要通过键盘输入命令来操作电脑，而不是用鼠标点击。

4. 实例演示：

  • 打开终端： 在Windows系统中，你可以打开“命令提示符”或者“PowerShell”；在macOS中，你可以打开“终端”应用；在Linux系统中，你可以打开“终端”。
  • 简单命令：
    • ls（在Linux和macOS上）或 dir（在Windows上）：列出当前目录中的文件和文件夹。
    • cd 目录名：改变当前工作目录。
    • python：进入Python交互模式，可以直接输入Python代码并执行。

5. 为什么学习终端：

使用终端可以让你更直接、更灵活地控制和管理你的电脑。很多高级的编程和开发工具都是通过终端来运行的，学会使用终端对成为一名程序员非常有帮助。

通过这章的学习，你应该对基本的电脑操作有了一定的了解。希望这些知识能帮助你更好地使用电脑！
